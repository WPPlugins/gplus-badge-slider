== Installation ==

This section describes how to install the plugin and get it working.

1.First Install our plugin. Make sure you have activated it.

This plugin is very easy to use. All you will need is your Google Plus Profile ID.

2.You can install it via Wordpress Installer or can download the file first then Upload it on /wp-content/plugins/ directory

3. You will see google plus badge settings from right side of your wordpress menu.

Configuration Options:

Google plus Id : Put your Google plus id - it normally visible on your profile URL - Something like 101560853443212199687

Example: https://plus.google.com/101560853443212199687/posts
